# JZON3

This is simple and fast JSON parser. The most importing thing is that handles cycles.
All sorts of types handles very fast.

## Installation for GitHub repo

 - Run "npm install"
 

## Usage
  
 - JZON3.stringify(obj);
 - JZON3.parse(string);
